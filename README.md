# Birthday Cake Project

A 3D interactive birthday cake experience built with Three.js.

## Deployment

This project is deployed on GitHub Pages. The site will be available at:
`https://[your-github-username].github.io/[repository-name]/`

## Project Structure

```
public/
  ├── models/      # 3D models
  ├── music/       # Audio files
  ├── src/         # Source code
  ├── cake.html    # Main page
  └── index.html   # Redirect page
```

## Local Development

1. Clone the repository
2. Open `public/cake.html` in your browser
3. For development, you can use a local server like `live-server` or `http-server`
